# Guides Not Included (archived)

An archived, Markdown copy of [**guidesnotincluded.com**](https://web.archive.org/web/*/guidesnotincluded.com)
— a site of *Oxygen Not Included* builds, guides, and resources ("Keep calm and
press the space bar"). The original site is offline; this repository restores
its content from the Internet Archive and serves it as an
[Astro](https://astro.build/) + [Starlight](https://starlight.astro.build/) site
on GitHub Pages.

## Quick start

The archived content is already generated and committed under
`src/content/docs/guidesnotincluded_archive/`, so running the site is just:

```bash
# Preview locally (Node 20+, pnpm 11 — `corepack enable` picks up the version
# pinned in package.json's "packageManager" field).
pnpm install
pnpm dev           # http://localhost:4321/guidesnotincluded/

# Build the static site (outputs ./dist, gitignored).
pnpm build

# Commit and push to main; GitHub Actions deploys to Pages.
```

### Refreshing the content (optional — rarely needed)

The crawler already ran and its output is committed, so you don't need to
re-generate anything. Re-run it only to pull fresh captures from the Wayback
Machine (needs internet access to `web.archive.org`):

```bash
pip install -r scripts/requirements.txt
python3 scripts/crawl_archive.py --domain guidesnotincluded.com
```

See **[RUNBOOK.md](RUNBOOK.md)** for the full crawl procedure and how to enable
network access and GitHub Pages.
