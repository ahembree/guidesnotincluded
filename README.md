# Guides Not Included (archived)

An archived, Markdown copy of [**guidesnotincluded.com**](https://web.archive.org/web/*/guidesnotincluded.com)
— a site of *Oxygen Not Included* builds, guides, and resources ("Keep calm and
press the space bar"). The original site is offline; this repository restores
its content from the Internet Archive and serves it as a
[MkDocs Material](https://squidfunk.github.io/mkdocs-material/) site on GitHub
Pages.

## Layout

| Path | What it is |
| --- | --- |
| `docs/` | The Markdown content (the site). Generated by the crawler. |
| `docs/assets/` | Images downloaded from the archived pages. |
| `mkdocs.yml` | MkDocs Material site configuration. |
| `scripts/crawl_archive.py` | Crawls the Wayback Machine and converts pages to Markdown. |
| `scripts/requirements.txt` | Python deps for the crawler. |
| `requirements.txt` | Python deps for building the site (MkDocs). |
| `.github/workflows/deploy.yml` | Builds and deploys the site to GitHub Pages on push to `main`. |
| `RUNBOOK.md` | Step-by-step procedure to (re)generate the content. |
| `LICENSE` | Licensing & attribution (CC BY-NC-SA 4.0 for the guide text). |
| `docs/attribution.md` | The on-site attribution & licensing page. |

## Quick start

```bash
# 1. Crawl the archive and convert to Markdown (needs internet access to
#    web.archive.org — see RUNBOOK.md if you're on Claude Code on the web).
pip install -r scripts/requirements.txt
python3 scripts/crawl_archive.py --domain guidesnotincluded.com

# 2. Preview the site locally.
pip install -r requirements.txt
mkdocs serve

# 3. Commit and push to main; GitHub Actions deploys to Pages.
```

See **[RUNBOOK.md](RUNBOOK.md)** for details, including how to enable network
access and GitHub Pages.
