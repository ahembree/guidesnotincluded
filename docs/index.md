# Guides Not Included

> Keep calm and press the space bar.

This site is an archived copy of **guidesnotincluded.com**, a collection of
*Oxygen Not Included* builds, guides, and resources. The original site is no
longer online; this copy was restored from the
[Internet Archive](https://web.archive.org/web/*/guidesnotincluded.com).

!!! note "Restore in progress"
    This is a placeholder home page. The archived guides have not been
    crawled yet. Run the crawler to populate the site:

    ```bash
    pip install -r scripts/requirements.txt
    python3 scripts/crawl_archive.py --domain guidesnotincluded.com
    ```

    See [`RUNBOOK.md`](https://github.com/ahembree/guidesnotincluded/blob/main/RUNBOOK.md)
    for the full procedure. The crawler will overwrite this page with the
