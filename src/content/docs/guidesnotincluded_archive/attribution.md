[guidesnotincluded.com](https://web.archive.org/web/2025/https://www.guidesnotincluded.com/),

[Internet Archive](https://web.archive.org/) and reformatted from the original

components (calculators, build planners) — is original work by the project
owner under the separate **MIT License**, independent of the content licence.

The split is drawn by file, so it's always clear which applies: a guide page
stays CC even when it loads a widget — it just references an MIT code file. A
component that reproduces the author's actual words or data keeps that
reproduced content under CC. Interactive additions still keep each page's
attribution credit and stay non-commercial.
